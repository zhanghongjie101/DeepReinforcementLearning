**摘要**: 并行分布式技术能有效加速深度强化学习的训练，现有并行分布式技术在强化学习领域仍有许多问题需要解决，比如如何加速单个智能体的采样，如何利用DRL阶段性特征加速训练，如何利用集成学习技术实现扩展性更强的DDRL训练框架。快速的DRL训练能加快强化学习新算法的研究和迭代，以及强化学习应用开发。
